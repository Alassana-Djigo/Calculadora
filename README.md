# Calculadora
Clonando a calculadora do iphone com dart e Flutter

## Pré-requisitos  
- Flutter (versão >= 2.x)  
- Dart (instalado juntamente com o Flutter)  

## Instalação  
Após clonar o repositório, navegue até o diretório do projeto e execute:  
```bash  
flutter pub get
